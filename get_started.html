Installation

Just follow 5 simple steps to add the SimplyClick editor to your template.

  1. Paste the SimplyClick files to your template. Don't forget to maintain the directory structure.

    ├── simply
|   └── css
|       └── myEdit.css
|   └── js
|       └── simply.js
├── css
|   └── your_style.css
|── js
|   └── your_javascript.js
└── index.html

  2. Add the css stylesheet link to your html document

    <link rel="stylesheet" href="simply/css/myEdit.css" type="text/css">

  3. Add the js file link to your html document

    <script src="simply/simply.js"></script>

  4. Add the save button to your html document

    <div id="clickme" class="glyphicon glyphicon-save"></div>
